Dimension.JS

A JavaScript framework for single page applications, DOM manipulation, and AJAX

DOM Manipulation

Dimension uses JavaScript objects to create and manipulate a virtual DOM, then pushes it into the browser DOM when you tell it to.

AJAX

Dimension has functions for POST and GET requests so you can easily update your SPA without reloading.

DML

DML (Dimension Module Loader) uses GET requests to load modules. Modules can be found on the creator's website.
